Otimo:
    O algoritmo ótimo funciona em duas execuções:
        Uma para gerar o json que indica os indices que cada página é referenciada;
        Uma para rodar o algoritmo ótimo de fato.
    Primeiro rode otimoReferencias.py com o arquivo de entrada que deseja e depois use o json gerado no otimo.py
    O arquivo de entrada descreve as páginas em ordem que são referenciadas

NFU:
    Para o NFU foi considerado que cada referenciamento a uma página é uma interrupção de relógioS
